Gravity Forms Range Slider  - WordPress plugin

== Description ==

The Gravity Forms Range Slider adds a dual handle range slider to Gravity Forms.

== Installation ==

Installing "Gravity Forms Range Slider" can be done by using the following steps:
1. Download the plugin after purchase
2. Upload the ZIP file (gravityformsrangeslider.zip) through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
3. Activate the plugin through the 'Plugins' menu in WordPress
== Frequently Asked Questions ==
= What does this plugin do? =
This plugin adds a Range Slider option to the Advanced Fields of Gravity Forms.
